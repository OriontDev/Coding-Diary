Saya adalah murid SMK Negeri 1 Denpasar, Dengan jurusan Rekayasa Perangkat Lunak atau RPL/PPLG. Saya masuk jurusan RPL karena saya menggemari Coding dan Game developing

Saya adalah anak kedua, Saya punya kakak dan adik laki-laki. Kakak saya juga alumni SMKN 1 Denpasar dan dia mengambil jurusan Multimedia.

Saya senang coding/mengembangkan game di waktu senggang, saya menggunakan software Construct 3 untuk membuat game. Selain Coding saya juga gemar bermain video game, bermain alat musik seperti biola. saya juga gemar Cosplay dan menonton anime, Saya senang berpergian dan jalan-jalan keluar kota/daerah/negara.

Saya memilih SMK Negeri 1 Denpasar karena itu adalah satu satunya SMK negeri yang mempunyai jurusan RPL. Saya bercita cita menjadi game developer dan sejak kecil saya sering memakai Roblox Studio dan Construct 2. Setelah lulus SMK saya akan mengikuti tes IELTS dan melanjutkan kuliah di Luar Negeri, maka dari itu mulai sekarang saya sudah mengikuti persiapan tes IELTS tersebut.
